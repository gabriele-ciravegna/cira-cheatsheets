Obsidian Skills kepano

Agent Skills per Obsidian. Permette a Claude Code / Codex / OpenCode di leggere e scrivere markdown Obsidian con wikilink, embed, callout, properties, bases, canvas.

Quando usarlo: usi Obsidian come vault note + Claude Code per editing. Coach Daily Cowork + assistant scrivono daily note in vault. Defuddle per estrarre markdown pulito da web pages.
NON usarlo se: Non usi Obsidian. Per markdown vanilla, skill stock di Claude Code basta.

Quick Start

# Marketplace Claude Code
/plugin marketplace add kepano/obsidian-skills
/plugin install obsidian@obsidian-skills

# Oppure npx skills
npx skills add git@github.com:kepano/obsidian-skills.git

# Manuale: aggiungi cartella .claude nel root del vault
git clone https://github.com/kepano/obsidian-skills.git
# poi copia skills/ in vault/.claude/skills/

Segue Agent Skills specification — funziona con qualsiasi agent compatibile.

Install per agent

  • Claude Code: /plugin install obsidian@obsidian-skills oppure cartella .claude/ nel root del vault
  • Codex CLI: copia skills/ in ~/.codex/skills/
  • OpenCode: git clone intera repo in ~/.opencode/skills/obsidian-skills (NO solo skills/ inner!)

Gotchas & Anti-pattern

  • OpenCode install: NO copiare solo skills/ — clona repo intera o auto-discovery fallisce.
  • Wikilink [[file]] case-sensitive su filesystem Linux. Su macOS HFS+ case-insensitive ma errore latente se sposti il vault.
  • Properties YAML in head note — formato strict. Skill obsidian-markdown conosce le regole, ma Edit raw può rompere.
  • Bases (.base) feature beta Obsidian — verifica versione Obsidian installata.
  • Defuddle richiede CLI separato installato. npm i -g @kepano/defuddle.
