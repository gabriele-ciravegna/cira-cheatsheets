Higgsfield Skill CC BY 4.0

Un MCP, 30+ modelli image/video. Skill Claude Code che wrappa il Higgsfield MCP — Seedance 2.0, Sora 2, Veo, Kling, Nano Banana Pro, GPT Image 2, Flux, Hailuo + Higgsfield exclusive Soul e Cinema Studio.

Quando usarlo: hai subscription Higgsfield e vuoi attivare modelli image/video con prompt naturale. Storyboard animation, poster, cinematic shot, b-roll.
NON usarlo se: NON hai subscription Higgsfield. La skill ha decision rule built-in: suggerisce solo se sei già subscriber. NON abbonarsi per un progetto solo.

Quick Start

# Clone skill
git clone https://github.com/robonuggets/higgsfield-skill

# Install MCP user-scope (una auth per tutte le sessioni)
claude mcp add --transport http --scope user higgsfield https://mcp.higgsfield.ai/mcp

# OAuth
/mcp
# Sign-in con account Higgsfield
# Restart Claude Code se tools non appaiono

Skill discovera .claude/skills/higgsfield/SKILL.md automatic.

Modelli inclusi (30+)

  • Video: Seedance 2.0, Sora 2, Veo, Kling, Hailuo, Higgsfield Cinema Studio (exclusive)
  • Image: Nano Banana Pro, GPT Image 2, Flux, Higgsfield Soul (exclusive)
  • Mixed: cinematic ad, b-roll, storyboard frame animation

One auth, no per-provider API key da gestire. Pricing transparency: skill mostra credit cost + cost-approval gate prima di run paid.

Gotchas & Anti-pattern

  • Sub Higgsfield obbligatoria — autore stesso dice "NON abbonarsi solo per un progetto, il valore è sbloccare i modelli che già paghi".
  • Restart Claude Code dopo MCP add — tools non appaiono altrimenti.
  • OAuth flow via /mcp — segui prompt sign-in browser.
  • Pricing trasparente ma cumulativo — monitora credit cost via Codeburn (anche se Codeburn potrebbe non aver supporto Higgsfield nativo).
  • Output video pesanti — pianifica storage + GDrive upload per condividere.
