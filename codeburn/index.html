Codeburn npm

"See where your AI coding tokens go." Tracker locale di token, cost, performance su 19 AI coding tool. Breakdown per task type, model, tool, project, provider.

Quando usarlo: mensilmente per audit costi AI coding. Capire se Opus vs Sonnet ti conviene. Confrontare provider. Identificare project che brucia di più.
NON usarlo se: Usi un solo tool (Claude Code) e costi non sono problema. Codeburn brilla con stack multi-tool (Claude + Codex + Cursor + altri).

Quick Start

# Install
npm install -g codeburn
# o brew
brew tap getagentseal/codeburn
brew install codeburn

# Run
codeburn                              # dashboard interattiva (7 giorni)
codeburn today
codeburn month
codeburn report -p 30days
codeburn report -p all
codeburn report --from 2026-04-01 --to 2026-04-10
codeburn status                       # one-liner today + month
codeburn export                       # CSV today/7d/30d
codeburn export -f json

Tutto locale. NO proxy, NO wrapper, NO API key. Legge session data direttamente da disco, pricing via LiteLLM.

Cosa faOne-liner: today + month spending. JSON disponibile.
QuandoStatus bar terminale, daily heartbeat.

Cosa faCSV con today/7d/30d. -f json per JSON.

App menu bar macOS disponibile separatamente — icona vicino orologio, status compatto.

Cosa traccia

  • 19 AI coding tool: Claude Code, Cursor, Codex, OpenCode, Aider, e altri
  • Per task type: edit, search, plan, review, debug
  • Per model: Opus 4.7, Sonnet 4.6, Haiku 4.5, GPT-5, Gemini, etc.
  • Per tool / project / provider: vedi quale cartella + tool + provider brucia di più
  • Optimize tab: suggerimenti su switch modello/tool
  • Compare tab: A/B tool su stesso task

Cursor + OpenCode richiedono better-sqlite3 (installato auto come optional dep).

CIRA Context

CIRA Use

Audit mensile spending

CIRA: usa codeburn month il primo di ogni mese per audit. Identifica project che brucia (es. dashboard build, coach Telegram heavy session). Decidi se switchare modello.

CIRA Stack

Vede anche Codex Plugin

Se usi Codex Plugin for Claude Code, Codeburn vede entrambi i provider (Anthropic + OpenAI) — utile per capire dove fluisce la spesa cross-modello.

CIRA Privacy

Tutto locale

NO API key richieste, NO proxy, NO upload. Legge JSONL session su disco. Nessuna telemetria fuori.

Gotchas & Anti-pattern

  • Pricing via LiteLLM — può lagging quando OpenAI/Anthropic cambiano prezzo. Aggiorna codeburn periodico.
  • Sessioni precedenti install di codeburn non tracciate — solo da install in poi.
  • Node 20+ required. Verifica node -v.
  • Menu bar app è separata da CLI — install entrambi se vuoi entrambi.
  • Cursor support richiede SQLite native module — possibili problemi build su M1/M3 (di solito si autoinstalla).
