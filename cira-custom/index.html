Skill custom CIRA Management — orchestra router + costruiamo build + mentori + brand

Quando usarlo: Lavoro CIRA quotidiano: routing decisioni multi-skill, build via /costruiamo, coaching mentori
Skip quando: Code review puro (→ gsd/ce), bug fix isolato

Quick Start

Orchestra-first rule

Su query multi-aspetto (emotion+context+decision) attiva orchestra PRIMA, mostra rete neurale completa, aspetta conferma. NO singola skill mentore.

/costruiamo trigger

User: "costruiamo X"
→ orchestra Step 1
→ brandkit Step 0.5
→ template-check Step 1.5
→ intervista 10-15 domande
→ path A1 (GSD) | A2 (gstack) | A3 (superpowers) | A4 (ce) | A5 (ruflo) | mix
→ execute con checkpoint

Router & Meta

orchestra + skill-creator + rete-neurale

orchestra
Cosa fa Rituale di apertura OBBLIGATORIO su ogni task complesso — scansiona risorse candidate, mostra longlist + shortlist + scartato con motivi, annuncia sequenza operativa, e ATTENDE CONFERMA prima di muovere.
Quando Query multi-aspetto, coaching personale, routing skill
Esempio User: 'sono bloccato su pricing' → orchestra invoca aristotle + hormozi + ali-pricing
See also /costruiamo (per build), rete-neurale-da-fonte
Orchestra-first rule: attiva PRIMA su query multi-aspetto, mostra rete, aspetta conferma
skill-creator-cira
Cosa fa Crea, modifica e fa evolvere skill per il sistema CIRA Management. Fonde il core Anthropic skill-creator con i 3 moduli del neural network CIRA (wikilinks, anti-conflitto, auto-aggiornamento) in una checklist canonica di 8 moduli — M1-M8.
Quando Creare/modificare skill CIRA-ready (M1-M8 checklist)
Esempio 'crea skill per X'
See also rete-neurale-da-fonte (wikilinks)
Ogni skill nuova CIRA aggiunta a MR immediately dopo verify-skill.py exit 0
rete-neurale-da-fonte
Cosa fa Meta-skill che trasforma una fonte (libro, canale YouTube, corpus multi-script) in skill CIRA-ready M1-M8 integrate nella rete neurale. Framework a 7 sessioni A-G con orchestrazione agenti.
Quando Estrarre rete neurale skill da corpus testuale, pipeline automatica
Esempio Trasforma canale YouTube in suite di skill integrate
See also skill-creator-cira, knowledge-base-builder

Build Orchestration

costruiamo + orchestratore-siti + app-builder + saas-mvp-launcher

/costruiamo X
Cosa fa Flow CIRA per costruire qualcosa di nuovo — sito, app, agente Telegram, MVP, sistema. v2.5.0 INTERVIEW-FIRST: nessun framework default, routing emerge da intervista lunga (10-15 domande) → 1-3 path proposti con razionale → utente sceglie → annuncio step+comandi → execute con checkpoint.
Quando Inizio progetto nuovo. Trigger: 'costruiamo X', 'facciamo l app', 'lets build'
Esempio /costruiamo agente telegram per gestione email
See also orchestra (Step 1), brandkit (Step 0.5), /gsd-* (path A1), /gstack:* (path A2), superpowers (path A3), ruflo (path A5)
v2.5.0 INTERVIEW-FIRST — nessun framework default, intervista 10-15 domande obbligatoria
orchestratore-siti
Cosa fa Layer specializzato PRE-/costruiamo. Si attiva quando il task riguarda un sito (landing, ecommerce, dashboard pubblico, redesign). Carica la knowledge base canonica CIRA per siti, impone Phase 1 PRE-BUILD QUESTIONS, sceglie la combo design in COMPATIBILITY-MATRIX, POI delega a /costruiamo.
Quando Build sito multipagina, design+content, landing page
Esempio User: 'costruiamo sito'
See also app-builder, /costruiamo, designlang
app-builder
Cosa fa Orchestratore che da una richiesta in linguaggio naturale porta a una app full-stack: classifica il tipo di progetto (SaaS, landing, mobile, CLI, API...), sceglie lo stack, scaffolda la directory, coordina agenti specializzati.
Quando Costruire applicazione web/mobile end-to-end, full-stack da zero
Esempio User: 'costruisci app'
See also saas-mvp-launcher, /costruiamo
saas-mvp-launcher
Cosa fa Roadmap operativa per portare un'idea SaaS a MVP in produzione in 4-6 settimane. Copre il cantiere intero — validazione, scelta stack moderno 2026, schema DB multi-tenant, auth via provider, pagamenti Stripe, deploy, checklist pre-launch.
Quando Lanciare MVP SaaS con stack predefinito, roadmap operativa
Esempio User: 'avvia SaaS'
See also app-builder, /costruiamo

Quando Mentore Marc Randolph — scrappy MVP, test-driven, nobody knows anything

Quando Cultura aziendale come filtro — Bezos/Jobs/Randolph/Branson/Marchionne/Musk

See also aristotle-first-principles, cira-diagnosis-protocol

Pensiero Strategico

aristotle + sinek

aristotle-first-principles
Cosa fa First principles strategic deconstruction for when you're stuck and don't know why. Strips away inherited assumptions, finds irreducible truths, and rebuilds solutions from zero.
Quando Decostruzione strategica bloccato + perché, quando il convenzionale gira a vuoto
Esempio User: 'sono bloccato su pricing, ci giriamo intorno' → invoca
See also founders-first-principles
NON usare per task tecnico — solo strategico/decisionale
sinek-golden-circle
Cosa fa Framework Simon Sinek per la comunicazione WHY-first e la leadership ispirante. Comunica dall'interno verso l'esterno, partendo dal WHY.
Quando Why-How-What positioning brand/offer, leadership ispirante
Esempio User: 'non so come comunicare il valore di quello che faccio'
See also godin-brand-recognition, merenda-positioning

Diagnostics

cira-diagnosis-protocol + network-functional-probe

cira-diagnosis-protocol
Cosa fa Protocollo di diagnosi strategico CIRA Management — analizza ogni nuovo prospect/cliente con 11 framework (McKinsey, Porter, Aristotele, Greene, Kahneman, Aurelio, Finance, Stochastic) in sequenza rigida e genera un Diagnosis Document PRIMA di ogni offerta.
Quando Pre-diagnosi cliente: recap → competitor → market → synthesis
Esempio Nuovo cliente. User: 'diagnosi cliente'
See also competitor-research-pipeline, cira-diagnosis-protocol
network-functional-probe
Cosa fa Probe funzionale settimanale della rete neurale CIRA. Lancia 10 query realistiche del lavoro Gabriele e verifica che le skill attese si attivino davvero via match deterministico sui trigger frontmatter. Output verde/giallo/rosso.
Quando Probe settimanale per verificare routing skill e activation della rete neurale
Esempio 'lancia network probe'
See also cira-diagnosis-protocol
