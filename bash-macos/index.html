Navigazione filesystem 3 comandi

pwd
Stampa working directory corrente
Verificare dove sei in filesystem. Essential per debugging path.
pwd

Vedi anche: cd, pushd/popd
cd
Cambia directory corrente

cd .. multipli sono lenti — usa cd /percorso/assoluto
Vedi anche: pwd, pushd/popd

-la dettagli + hidden. -lh dimensioni leggibili. -t ordinamento data.
ls -lah ~/WORK

Vedi anche: tree, find, exa (alternativa moderna)

Operazioni su file 7 comandi

mkdir

-p crea parent directories se non esistono.
mkdir -p ~/WORK/SITI/ispiration/sitename/{firecrawl,designlang}
Vedi anche: rmdir, rm -rf
cp

Vedi anche: mv, rm
mv
Sposta o rinomina file
Funziona anche tra filesystem (a differenza di Linux).

Vedi anche: cp, rm
rm
Cancella file
-r ricorsivo per cartelle. -i chiede conferma. NO -f senza -i.

IRREVERSIBILE. Non c'è undelete macOS. Usa Finder/Trash prima di rm
touch
Crea file vuoto o aggiorna timestamp
Creare file marker. Aggiornare mtime senza editare.

find
Cerca file ricorsivamente per nome/tipo/data/dimensione
-name pattern. -type f/d. -mtime -N giorni. -size +10M.
find ~/WORK -name '*.md' -mtime -7 -type f
~/WORK/_AWARENESS/2026-05-14-HANDOFF-pre-clear.md
Da '.' non da '/' — altrimenti scansiona FS intero. Usa find ~/WORK non find /
Vedi anche: fd (alternativa veloce), grep -r
fd
Alternativa moderna e veloce a find
PREFERISCI A FIND. 10× più veloce. Regex di default. (brew install fd)
fd '\.md$' ~/WORK --changed-within 7d
Vedi anche: find (POSIX originale)

Ricerca e manipolazione testo 10 comandi

cat
Stampa contenuto file
File piccoli. Per editor: nano/vim. Per leggere in script: $(cat file)
cat ~/.zshrc | head -20
Vedi anche: less, head/tail
head / tail
Prime/ultime N righe file
-n N righe. tail -f segue log in real-time.
head -20 ~/WORK/_AWARENESS/HANDOFF-2026-05-14.md
Vedi anche: less (viewer interattivo)
grep
Cerca testo in file
-i case-insensitive. -v inverti (NOT). -n numeri riga.
grep -n 'TODO' ~/WORK/SCRIPTS/aurelio-bot-launcher.sh
grep -rE
Cerca regex ricorsivo nei file
-r ricorsivo. -E regex extended. -l solo nomi file. --include='*.py'
grep -rE 'TODO|FIXME' ~/.claude/skills --include='*.md'
Vedi anche: rg (ripgrep — 10× più veloce)
rg
ripgrep — search ricorsivo ultra-veloce con regex
PREFERISCI A GREP. 10× più veloce. -i case-insensitive. -w word boundary. -A 3 contesto.
rg 'costruiamo' ~/.claude/skills -A 2 -B 2
Install: brew install ripgrep
sed
Stream editor — modifica testo da CLI
-i in-place. s/old/new/g replace all. d cancella righe.
sed -i '' 's/OLD_URL/NEW_URL/g' *.md
macOS: sed -i '' (empty string dopo -i). Linux: sed -i no spazi
awk
Data processing — split righe, stampa colonne
Estrarre colonne da output. '{print $1,$3}'. -F delimiter.
ps aux | awk '{print $1,$11}' | head
Vedi anche: cut, paste
cut
Estrai colonne fisse da righe
-d delimitatore. -f 1,3 campi 1 e 3.
ps aux | cut -d' ' -f1-2
sort / uniq
Ordina righe / rimuove duplicati
sort -n numerico. sort -r reverse. uniq -c conta occorrenze.
cat file.txt | sort | uniq -c | sort -rn
42 skill-name 38 another-skill
wc
Word/line/byte count
-l righe. -w parole. -c byte.
find ~/.claude/skills -name 'SKILL.md' | wc -l
2577
diff
Confronta due file
-u unified format. -r directory ricorsivo.
diff -u old.json new.json | head -50

Rete 6 comandi

curl
HTTP client per request da terminale
-s silent. -S show errors. -m timeout. -o file. -w format.
curl -sS -m 5 -o /dev/null -w 'HTTP:%{http_code}\n' http://localhost:37777
HTTP:200
Sempre -sS (silent ma error). Timeout critico: -m 5 secondi
Vedi anche: wget, httpie
curl POST/JSON
HTTP POST con JSON payload
API POST, PUT, DELETE.

Vedi anche: jq per costruire JSON dinamico
wget
Download file da URL
-O salva con nome. -q quiet. -t 3 retry. (brew install wget)

scp

Processi e launchd 4 comandi

ps
Lista processi in esecuzione
Trovare PID di un processo. aux full listing.
ps aux | grep coach-bot
cira 2847 0.2 1.3 12345 98765 ?? S 2:42PM bun /Users/cira/.claude/coach-bot.ts
kill
Termina processo per PID
-9 force (SIGKILL). -15 graceful (SIGTERM, default).
kill -15 2847
CIRA: MAI pkill un servizio launchd — usa launchctl. kill riserva per debug
launchctl
Gestione servizi launchd macOS (daemon, agent)
Avviare/fermare servizio permanente. Verificare stato com.cira.* coach-bot, Aurelio.
launchctl list | grep cira
- 0 com.cira.coach-bot
# Load service
launchctl bootstrap gui/$(id -u) \
  ~/Library/LaunchAgents/com.cira.coach-bot.plist

# Unload service
launchctl bootout gui/$(id -u) \
  ~/Library/LaunchAgents/com.cira.coach-bot.plist
MAI pkill un servizio launchd — hai rotto MCP e dovrai restartare Claude Code
launchctl print
Stampa dettagli servizio launchd
Vedere config, env vars, status di un servizio.
launchctl print gui/$(id -u)/com.cira.coach-bot

Redirect e pipe patterns 9 pattern

SintassiCosa faQuando / Note
cmd > file stdout in file (OVERWRITE) Salvare output. Attenzione: sovrascrive!
cmd >> file stdout in file (APPEND) Aggiungere a file. Non sovrascrive. Sicuro.
cmd 2> file stderr in file Catturare errori — es. curl bad-url.com 2> error.log
cmd 2>&1 stderr → stdout Combine error+output — es. script.sh 2>&1 | tee log.txt
cmd1 | cmd2 Pipe stdout di cmd1 in stdin di cmd2 Incatenare — es. cat file | grep 'error' | wc -l
cmd | tee file Sia terminale che file Salvare E vedere output — es. curl api.com | tee response.json
xargs -n1 cmd Esegui cmd per ogni riga input Costruire comandi dinamici — es. find . -name '*.tmp' | xargs rm
cmd <<< 'string' Here-string in stdin Passare variabile come input — es. grep 'error' <<< "$log"
cmd << EOF ... EOF Here-doc multiriga Creare file inline. Gotcha: EOF senza spazi. EOF tra apici = literal (no var expansion)

macOS Gotchas

⚠
sed -i '' su macOS Sed syntax macOS: sed -i '' (empty string dopo -i). Linux usa sed -i senza spazi — non sono compatibili.
⚠
Cron PATH non eredita .zshrc Crontab NON eredita PATH e variabili da ~/.zshrc. Usa sempre percorsi assoluti nei cron job (es. /usr/local/bin/node non node).
🔴
MAI pkill su servizi launchd Processi launchd (Aurelio, claude-mem): usa sempre launchctl bootout per fermare, mai pkill. pkill ha rotto MCP e ha richiesto restart Claude Code.
⚠
pbcopy e secret Dati copiati con pbcopy entrano nel clipboard di sistema — mai copiare token, API key, o password con pbcopy.
⚠
gws drive — percorsi assoluti rifiutati gws drive files create --upload richiede cd nella directory del file prima. I path assoluti vengono rifiutati con validationError.
⚠
Timezone cron — America/Los_Angeles default La timezone di cron su macOS è America/Los_Angeles di default. Se serve un fuso orario diverso, specificalo nella configurazione launchd con <key>TimeZone</key>.
